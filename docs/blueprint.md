# **App Name**: 宏量助手

## Core Features:

- 食物录入: 用户可以通过搜索数据库或提供详细描述来输入所消耗的食物。
- AI食物描述解析器: 利用AI工具解析自由形式的食物描述（例如，“一个小苹果”，“150克鸡胸肉”），并推荐具体的食物项目，提供估计的数量和宏量营养素分解，供用户确认。
- 营养数据集成: 连接外部营养数据库API，获取已录入食物的详细蛋白质和碳水化合物信息。
- 每日摄入计算: 根据所有已记录的食物项目，自动计算并汇总当天蛋白质和碳水化合物的总摄入量。
- 每日摘要显示: 清晰简洁地展示当天累计的蛋白质和碳水化合物总量概览。

## Style Guidelines:

- 主要颜色：平衡的中间绿色（`#339966`），唤起成长、清新和自然健康的感觉。它将为交互元素和品牌提供扎实的基础。
- 背景颜色：柔和的、接近白色的绿色（`#ECF5F0`），带有微妙的暖意，为内容显示创造一个干净、通透、平静的画布。
- 强调颜色：充满活力的柠檬绿色（`#7DD64C`），为行动召唤、进度指示器以及突出关键营养成就或数据点提供高对比度。
- 正文和标题字体：'Inter' (无衬线字体)，因其现代、清晰和客观的易读性，适用于各种字号，确保所有营养数据都易于解析。
- 图标：采用简洁、基于轮廓的图标，具有现代、友好的美感。图标应视觉化地代表常见的食物类型、宏量营养素类别（例如，蛋白质分子、碳水化合物插图）以及基本的UI操作（如“添加”或“搜索”）。
- 布局：对已记录的食物项目采用简洁的卡片式布局，使单个条目及其相关的宏量营养素易于理解。一个突出、响应式的仪表板将在屏幕顶部清晰显示每日总计。
- 动画：融入细微的动画以提供反馈，例如添加新食物项目时的轻微放大效果，或每日总计更新时平滑的进度条动画，在不分散用户注意力的情况下增强用户参与度。
